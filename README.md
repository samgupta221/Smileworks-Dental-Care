# 🦷 Smileworks Dental Care

A premium, elegant, and fully responsive dental clinic website built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. Designed with a modern healthcare aesthetic, smooth animations, interactive UI components, and conversion-focused sections to help dental clinics attract and engage patients.

---

## 🌐 Live Demo

🔗 https://fastidious-lollipop-1451a2.netlify.app/

---

## ✨ Features

- 🦷 Premium Modern Dental Clinic Design
- 📱 Fully Responsive Layout
- 🎨 Clean Luxury UI/UX
- 📅 Appointment Booking Form
- 👨‍⚕️ Meet the Doctors Section
- ⭐ Patient Testimonials Carousel
- ❓ Interactive FAQ Accordion
- 📞 Contact Information Section
- 📈 Animated Statistics Counters
- 🎯 Scroll Reveal Animations
- 📌 Sticky Navigation
- 📲 Floating Call & WhatsApp Buttons
- ⚡ Fast Loading & SEO Optimized
